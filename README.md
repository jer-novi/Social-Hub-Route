# Social-Hub-Route

Routeplanner voor bezorgshifts rondom The Social Hub Den Haag.

## Gebruik

1. Open `index.html` uit je lokale projectmap in je browser.
2. Stel de gewenste radius in (km).
3. Vink straten af in de lijst of klik markers op de kaart.
4. Kies de looprichting voor huisnummers (laag ➜ hoog of hoog ➜ laag).

De app toont per straat:
- of de straat binnen de ingestelde radius valt
- nummerbereik (tot welke nummers de straat loopt)
- actuele looprichting
